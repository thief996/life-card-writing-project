# 牌卡資料庫欄位 v2

正式牌庫需要區分卡牌系統，不能只用內容系列分類。

| 欄位 | 用途 |
|---|---|
| 卡牌名稱 | 編號與名稱 |
| 卡牌類型 | 人生卡牌、成就卡、隱藏獎勵、限定卡、概念草案 |
| 卡號 | 各系統內的編號 |
| DAY | IG 連載日 |
| 系列 | 人生考古學、童年寶物等內容分類 |
| 主標題 | 卡名 |
| 副標題／金句 | 最核心的一句 |
| 稀有度 | N、R、SR、SSR |
| 收藏日期 | 正式解鎖日期 |
| 獲得條件 | 觸發這張卡的真實條件 |
| 卡牌效果 | 這段故事留下的提醒 |
| 能力值 | 可包含多項能力 |
| 圖鑑狀態 | 正式收藏、候選、早期概念 |
| IG 狀態 | 撰寫中、設計中、待發布、已發布 |
| 完整故事 | 不應只放摘要 |
| 圖檔 | 正式封面與輪播素材 |
| 來源 | 對話、日記或原始素材連結 |

## 抽卡規則

「抽卡」預設只從 `圖鑑狀態＝正式收藏` 或明確建立的 `任務卡池` 中抽取。

早期概念、示範文案與未經玩家確認的題目不得混入正式抽卡。
