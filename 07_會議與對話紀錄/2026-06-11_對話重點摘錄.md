# 2026-06-11 對話重點摘錄

## 品牌方向

排除心靈成長老師風、文青療癒風與真實考古學風；選擇卡通、手繪、桌遊或冒險筆記、有世界觀、具收藏卡牌感的方向。

## 原始色彩提案

- 湖泊藍綠：主色
- 暖米白：背景
- 牛皮紙棕：故事與裝飾
- 墨灰：文字

後續回饋指出湖水色偏綠，正式版本需再往藍色調整。

## 三種核心風格

### Version A：冒險者圖鑑

關鍵字：RPG 圖鑑、冒險筆記、手繪地圖、卡牌收藏冊。

核心感受：「我在人生路上撿到一個故事，放進圖鑑裡。」

### Version B：暖呼呼發瘋研究室

關鍵字：手帳、塗鴉、筆記本、發瘋老師。

特色：親近、像朋友，容易累積信任感。

### Version C：人生卡牌博物館

關鍵字：收藏卡、溫暖插畫、略有塔羅結構但不神秘。

核心感受：「每個人都是一張值得收藏的人生卡。」

## 最終配比

- Version A：60%
- Version C：30%
- Version B：10%

形成「人生副本」世界觀、「冒險圖鑑」呈現、「卡牌收藏」互動，以及暖呼呼人生研究員的語氣。
